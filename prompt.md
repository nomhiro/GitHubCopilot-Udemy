# Marpスライド生成プロンプト

## 概要
このプロンプトを使用して、Udemy動画講座用のMarp形式スライドを生成してください。

---

## 指示

以下の条件に従って、指定されたセクションのMarp形式スライド（slide.md）を作成してください。

### 1. 基本設定

```yaml
---
marp: true
theme: custom-theme
paginate: true
---
```

- テーマは `custom-theme` を使用（themes/custom-theme.css）
- ページ番号を表示

---

### 2. スライドの種類と使い分け

#### (A) セクションタイトルスライド
- セクションの最初と最後に使用
- 下部に青い帯が表示される

```markdown
<!-- _class: title -->
<!-- _paginate: false -->

# セクションタイトル

## Section XX
```

#### (B) レクチャータイトルスライド
- 各Lectureの開始時に使用
- 左側に青い帯が表示される

```markdown
<!-- _class: lecture -->
<!-- _paginate: false -->

# Lecture X: レクチャータイトル
```

#### (C) 通常の説明スライド
- コンテンツを説明するスライド
- タイトル（h1）は上部に固定表示
- 特別な指定は不要

```markdown
# スライドタイトル

## 小見出し

- 内容1
- 内容2
```

---

### 3. デザインルール

#### 重要ポイントの強調（赤字）
試験ポイントや重要な注意点は赤字で強調：

```html
<span class="red">試験ポイント: ここに重要な内容</span>
```

#### 表の活用
情報を整理する場合は表を使用：

```markdown
| 項目 | 説明 |
|------|------|
| 項目1 | 説明1 |
| 項目2 | 説明2 |
```

#### コードブロック
コード例はシンタックスハイライト付きで表示：

```python
def example():
    return "Hello"
```

---

### 4. スライド構成

1. **セクションタイトルスライド**（_class: title）
2. **学習目標スライド**（通常スライド）
3. **Lecture 1 タイトル**（_class: lecture）
4. **Lecture 1 内容スライド**（通常スライド × 複数枚）
5. **Lecture 2 タイトル**（_class: lecture）
6. **Lecture 2 内容スライド**（通常スライド × 複数枚）
7. ... 繰り返し ...
8. **まとめスライド**（通常スライド）
9. **試験対策スライド**（通常スライド）
10. **セクション完了スライド**（_class: title）

---

### 5. コンテンツ作成のポイント

- **初心者にわかりやすい解説**を心がける
- **試験ポイント**は赤字で明示する
- **数値データ**（統計など）は表で整理し、強調する
- **キーボードショートカット**は表形式でWindows/Mac両方記載
- **よくある誤答**も紹介して、試験対策に役立てる
- 1スライドに情報を詰め込みすぎない

---

### 6. 参考情報の取得

必要に応じて以下のツールを使用：
- **Microsoft Docs MCP**: Microsoft Learn から最新情報を取得
- **Context7 MCP**: ライブラリのドキュメント取得
- **DeepWiki MCP**: GitHubリポジトリの情報取得
- **lectures.md**: 各セクションフォルダ内のレクチャー内容

## テーマのカスタマイズ

テーマを変更する場合は `themes/custom-theme.css` を編集してください。

### 主なスタイル

| クラス | 用途 | 特徴 |
|--------|------|------|
| `title` | セクションタイトル | 中央揃え、下部に青い帯 |
| `lecture` | レクチャータイトル | 左揃え、左側に青い帯 |
| （なし） | 通常スライド | 上部にタイトル固定、上揃え |

### カラー

- メインカラー（青）: `#4A90D9`
- 強調色（赤）: `#E53935`
- テキスト色: `#333`
