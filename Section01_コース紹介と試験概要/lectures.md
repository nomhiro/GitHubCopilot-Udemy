# コース紹介と試験概要

## レクチャー1: コース紹介と学習の進め方
**時間目安: 5分**

### 講座概要

本講座は、GitHub Copilot認定資格（GH-300）試験の合格を目指す方のための総合対策コースです。

試験ドメインの配点比率を考慮し、実践的なハンズオンを含めた構成となっています。

---

### 想定受講者

- GitHub Copilotを使った開発を学びたい方
- GH-300認定資格の取得を目指す方
- AI支援開発（AIペアプログラミング）に興味がある開発者
- 組織でのGitHub Copilot導入を検討している方
- プロンプトエンジニアリングを学びたい方

---

### 講座のロードマップ

**【入門編】**
- GitHub Copilotの概要と試験範囲
- 責任あるAIの原則
- プラン比較（Individual/Business/Enterprise）

**【活用編】**
- Copilot Chat & エージェントモード
- コーディングエージェント
- コードレビューとPull Request
- GitHub Copilot Spaces
- CLIでのGitHub Copilot

**【技術編】**
- GitHub Copilotのしくみとデータ処理
- プロンプト作成とプロンプトエンジニアリング
- AIの開発者ユースケース
- テスト生成

**【管理編】**
- プライバシーとコンテキスト除外
- セキュリティ・パフォーマンス・トラブルシューティング

**【試験対策】**
- 試験ドメイン別重要ポイント総復習
- 模擬試験（2回・計60問）

---

### 講座の注意事項

- 本講座の座学編は、AIで音声を作成しております
- 受講者の皆様に聞き取りやすく、一貫した声をお届けするためにAI音声を活用しています
- 内容と説明については、講師が厳密に作成しております
- 受講時にAI音声が好まれない場合は、購入後でもUdemy上で返金手続きが可能です

---

### 本講座の進め方

- 各セクションで、機能の解説などの「座学編」と、実際にVS Codeで実践する「ハンズオン編」があります
- 各セクション末には「確認テスト」を用意しており、理解度をチェックできます
- 最終セクションでは模擬試験（2回・計60問）で本番に向けた実力を測定できます
- ぜひ、ハンズオンで動作を確認して、皆様の業務への活用イメージを膨らませてください

---

### 講座の前提

以下の環境を用意し、実践して一緒に知見を身につけましょう！
※のちのセクションで詳しく解説し、構築方法もレクチャーいたします

- Visual Studio Code が使えること
- GitHubアカウントがあること
- GitHub Copilotライセンス（Individual/Business/Enterprise のいずれか）
