## レクチャー1: Copilotプランの全体像（Free/Pro/Business/Enterprise）
**時間目安: 12分**

### 学習目標
- GitHub Copilotの各プランの概要を理解する
- プランごとの対象ユーザーを把握する
- プラン選択の基本的な考え方を学ぶ

### 内容

#### GitHub Copilotの価格プラン

GitHub Copilotの開発においては、セキュリティ、プライバシー、コンプライアンス、透過性に重点が置かれています。すべての開発者と組織には特定のニーズがあるため、複数の価格プランが用意されています。

| プラン | 対象 | 特徴 |
|--------|------|------|
| **Free** | 個人開発者（入門） | 月2,000コード補完、50チャットメッセージの制限付き |
| **Pro** | 個人開発者 | 無制限のコード補完とチャット |
| **Business** | 組織・チーム | 管理機能、セキュリティ機能、IP補償 |
| **Enterprise** | 大規模企業 | Business機能＋カスタマイズ機能、ナレッジベース |

#### GitHub Copilot Free
- **対象**: GitHub Copilotを初めて試す個人開発者
- **使用制限**:
  - 月あたり2,000個のコードオートコンプリート
  - 月あたり50個のチャットメッセージ
- **開始方法**: Visual Studio CodeでGitHub Copilotアイコンをクリック → 「Sign in to Use GitHub Copilot for Free」

#### 無料で利用できるユーザー
以下のユーザーはCopilot Proを無料で利用可能：
- 教師
- 学生
- 選択されたオープンソースのメンテナー

詳細: https://aka.ms/Copilot4Students

## レクチャー2: Copilot Individualの機能と特徴
**時間目安: 10分**

### 学習目標
- Copilot Free/Proの機能を理解する
- 個人利用での制限事項を把握する
- Free と Pro の違いを説明できる

### 内容

#### Copilot Free/Proで利用可能な機能

| 機能 | Free | Pro |
|------|------|-----|
| コード補完 | 月2,000回まで | 無制限 |
| チャットメッセージ | 月50回まで | 無制限 |
| パブリックコードフィルター | ✅ | ✅ |
| Copilot拡張機能との統合 | ✅ | ✅ |
| 内部ツール用プライベート拡張機能の構築 | ✅ | ✅ |

#### Free/Proで利用できない機能

以下の機能はBusiness/Enterpriseプラン限定：
- ユーザー管理
- 既定でトレーニングから除外されるデータ
- エンタープライズグレードのセキュリティ
- IP補償
- コンテンツの除外
- SAML SSO認証
- 使用状況メトリック

#### パブリックコードフィルター
- すべてのプランで利用可能
- GitHubの公開リポジトリと一致するコード提案をフィルタリング
- ライセンス問題を回避するための重要な機能

## レクチャー3: Copilot Businessの機能と管理機能
**時間目安: 12分**

### 学習目標
- Copilot Businessの管理機能を理解する
- 組織でのポリシー管理方法を把握する
- セキュリティ機能の重要性を説明できる

### 内容

#### Copilot Businessの管理ポリシー機能

| 機能 | 説明 |
|------|------|
| **ユーザー管理** | 組織メンバーへのライセンス割り当てと管理 |
| **既定でトレーニングから除外されるデータ** | 組織のコードがAIモデルのトレーニングに使用されない |
| **エンタープライズグレードのセキュリティ** | 高度なセキュリティ対策の適用 |
| **IP補償** | 知的財産に関する法的保護 |
| **コンテンツの除外** | 特定のファイルやリポジトリをCopilot分析から除外 |
| **SAML SSO認証** | 企業の認証システムとの統合 |
| **使用状況メトリック** | 組織全体の使用状況を監視・分析 |

#### ポリシー管理の重要性
- **組織レベルでの一括管理**: チーム全体に一貫したポリシーを適用
- **コンプライアンス対応**: 組織のプライバシーポリシーに従った機密データの取り扱い
- **監査対応**: 詳細な監査ログへのアクセス

#### データプライバシーとセキュリティ
Copilot Businessでは、より堅牢なプライバシー制御を提供：
- GitHub Copilot分析から特定のファイルを除外する機能
- 詳細な監査ログへのアクセス
- IP補償の提供

## レクチャー4: Copilot Enterpriseの機能とナレッジベース
**時間目安: 15分**

### 学習目標
- Copilot Enterprise専用機能を理解する
- ナレッジベースの概念と利点を説明できる
- カスタマイズ機能の活用方法を把握する

### 内容

#### Copilot Enterprise専用のカスタマイズ機能

| 機能 | 説明 |
|------|------|
| **プライベートコードベースに合わせたチャット調整** | 組織固有のコードベースを理解した応答 |
| **ナレッジベースのアタッチ** | 組織のコンテキストに合わせたチャット応答 |

#### ナレッジベース機能
- **定義**: 組織固有の情報やドキュメントをCopilotに提供する仕組み
- **利点**:
  - 組織のコーディング規約に沿った提案
  - 内部ドキュメントを参照した回答
  - プライベートコードベースの理解

#### GitHub Enterprise Cloudの要件
- Copilot EnterpriseはGitHub Enterprise Cloudが必要
- 大規模組織向けの統合管理機能

#### カスタマイズ機能の比較

| 機能 | Free/Pro | Business | Enterprise |
|------|----------|----------|------------|
| プライベートコードベースに合わせたチャット調整 | ❌ | ❌ | ✅ |
| Copilot拡張機能との無制限統合 | ✅ | ✅ | ✅ |
| 内部ツール用プライベート拡張機能の構築 | ✅ | ✅ | ✅ |
| 組織のナレッジベースをチャットにアタッチ | ❌ | ❌ | ✅ |

## レクチャー5: 各プランの違い比較（データ除外・IP補償・請求）
**時間目安: 10分**

### 学習目標
- 全プランの機能を一覧で比較できる
- プラン選択時の重要な考慮事項を理解する
- 組織のニーズに適したプランを選択できる

### 内容

#### 管理ポリシー機能の比較

| 機能 | Free/Pro | Business | Enterprise |
|------|----------|----------|------------|
| パブリックコードフィルター | ✅ | ✅ | ✅ |
| ユーザー管理 | ❌ | ✅ | ✅ |
| 既定でトレーニングから除外されるデータ | ❌ | ✅ | ✅ |
| エンタープライズグレードのセキュリティ | ❌ | ✅ | ✅ |
| IP補償 | ❌ | ✅ | ✅ |
| コンテンツの除外 | ❌ | ✅ | ✅ |
| SAML SSO認証 | ❌ | ✅ | ✅ |
| GitHub Enterprise Cloud必須 | ❌ | ❌ | ✅ |
| 使用状況メトリック | ❌ | ✅ | ✅ |

※ GitHub Copilot Freeには使用制限があります

#### プラン選択時の重要な考慮事項

##### 1. データのプライバシーとセキュリティ
- プランごとに異なるレベルのデータプライバシーとセキュリティ対策
- より堅牢なプライバシー制御はBusiness/Enterprise限定
- 機密データを扱う組織はBusiness以上を検討

##### 2. ポリシー管理
- 組織レベルでのCopilotポリシー管理の必要性
- Business/Enterpriseで包括的なポリシー管理が可能
- 組織のプライバシーポリシーへの準拠

##### 3. データの収集と保持
- データプライバシー規制への準拠
- 個々のサブスクライバーはGitHubがプロンプトと提案を収集・保持するかどうかを選択可能

##### 4. IP補償とデータプライバシー
- Business/Enterpriseで提供されるIP補償
- 法律、セキュリティ、顧客の問題を回避するために重要
- 企業での利用には必須の考慮事項

## レクチャー6: 【ハンズオン】プランの選び方と導入手順
**時間目安: 8分**

### 学習目標
- 実際にプランを選択する際の判断基準を理解する
- 導入手順の流れを把握する
- 組織での展開方法を学ぶ

### 内容

#### プラン選択のフローチャート

```
開始
  │
  ├─ 個人での利用？
  │   ├─ Yes → 有料で使いたい？
  │   │         ├─ Yes → Copilot Pro
  │   │         └─ No  → Copilot Free
  │   │
  │   └─ No（組織利用）
  │         │
  │         ├─ ナレッジベースやカスタマイズが必要？
  │         │   ├─ Yes → Copilot Enterprise
  │         │   └─ No  → Copilot Business
  │         │
  │         └─ GitHub Enterprise Cloud利用中？
  │             ├─ Yes → EnterpriseまたはBusiness
  │             └─ No  → Business
```

#### 導入チェックリスト

**個人（Free/Pro）**
- [ ] GitHubアカウントの作成
- [ ] Copilotへのサインアップ
- [ ] IDEへの拡張機能インストール
- [ ] 初期設定の確認

**組織（Business/Enterprise）**
- [ ] 組織のニーズ分析
- [ ] プランの選択と契約
- [ ] 組織設定でのCopilot有効化
- [ ] ユーザーへのライセンス割り当て
- [ ] ポリシーの設定
- [ ] コンテンツ除外の設定（必要に応じて）
- [ ] 使用状況モニタリングの設定

#### よくある質問

**Q: Freeプランで十分ですか？**
A: 個人の学習目的や小規模なプロジェクトであれば十分です。ただし月の使用制限（2,000コード補完、50チャット）があります。

**Q: BusinessとEnterpriseの違いは？**
A: 主な違いはナレッジベース機能とプライベートコードベースへのカスタマイズ。大規模組織で独自のコーディング規約がある場合はEnterpriseが有効。

**Q: IP補償とは何ですか？**
A: Copilotが生成したコードに関連する知的財産権の問題に対する法的保護。Business/Enterpriseで提供。

### 問題14
GitHub Copilot Free/Proで利用できない機能はどれですか？

- A) パブリックコードフィルター

- C) 自己署名証明書によるVPNプロキシのサポート ✓

### 問題15
組織がGitHub Copilotのポリシーを一括管理する必要がある場合、最低限必要なプランはどれですか？

- A) Free
- B) Pro
- C) Business ✓
- D) プランに関係なく利用可能

> **解説**: 組織レベルでのポリシー管理、ユーザー管理、コンテンツ除外などの機能はBusiness以上のプランで利用可能です。Free/Proは個人向けのプランです。
