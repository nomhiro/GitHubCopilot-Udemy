## コース説明

GitHub Copilot認定資格（GH-300）に合格するための対策講座です。

本コースは、Microsoft Learn公式リソースと連携し、試験の7つのドメイン（責任あるAI、Copilotプラン・機能、データ処理の仕組み、プロンプトエンジニアリング、開発者ユースケース、テスト生成、プライバシーとコンテキスト除外）を網羅的に学習できます。

**【このコースで学べること】**
- GitHub Copilotの基本概念と各プラン（Individual/Business/Enterprise）の違い
- 責任あるAIの6つの原則とその実践方法
- IDEでのコード補完、インラインチャット、複数提案の活用
- GitHub Copilot Chat、エージェントモード、MCP Serverの使い方
- CLI環境でのGitHub Copilot活用
- 効果的なプロンプトの書き方（4つのSの原則）
- ユニットテスト・統合テストの自動生成
- プライバシー設定とコンテンツ除外の構成
- セキュリティ脆弱性の特定とトラブルシューティング

**【対象者】**
GitHub Copilot認定資格を取得したい方、AIを活用した開発スキルを身につけたい方、組織でのCopilot導入を検討している方に最適です。

**【前提知識】**
基本的なプログラミング経験とGitHub・VS Codeの基本操作ができれば受講可能です。

本コースを修了すれば、GH-300試験に自信を持って臨めるだけでなく、実務でGitHub Copilotを最大限に活用するスキルが身につきます。

---

## 学習内容（What you'll learn）

1. GitHub Copilot認定資格（GH-300）の試験範囲を完全に理解し、合格に必要な知識を習得できる
2. 責任あるAIの6つの原則（公平性、信頼性、プライバシー、包括性、透明性、説明責任）を理解できる
3. GitHub Copilot Individual/Business/Enterpriseの各プランの違いと選び方がわかる
4. IDE、Chat、CLIでGitHub Copilotを効果的に活用するスキルが身につく
5. プロンプトエンジニアリングの原則（4つのS）を使って効率的なコード生成ができるようになる
6. GitHub Copilotを使ったユニットテスト・統合テストの自動生成ができる
7. プライバシー設定とコンテンツ除外の設定方法を理解できる
8. 模擬試験2回分（計60問）で実践的な試験対策ができる
